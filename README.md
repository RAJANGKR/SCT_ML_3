# Cats and Dogs Classification using SVM

This project is part of my **SkillCraft Technology Machine Learning Internship - Task 3**.  
The goal of this project is to classify images of cats and dogs using a **Support Vector Machine (SVM)** model.

## Objective

To build an image classification model that can distinguish between **cats** and **dogs** using image data.

## Dataset

- **Dataset:** PetImages / Cats and Dogs dataset
- **Classes:** Cat, Dog

## Tools & Libraries

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Pillow
- Scikit-learn

## Workflow

- Loaded cat and dog images from folders
- Converted images to grayscale
- Resized images to a fixed size
- Flattened images into feature vectors
- Split data into training and testing sets
- Applied feature scaling
- Trained an SVM classifier
- Evaluated the model using accuracy and confusion matrix

- `ml3.ipynb` – Jupyter Notebook with full code
- `README.md` – Project documentation

## Conclusion

This project helped in understanding image preprocessing, feature extraction, and binary image classification using SVM.
